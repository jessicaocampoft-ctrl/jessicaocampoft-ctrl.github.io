# Instrucciones del proyecto

- Siempre que una tarea modifique `google-apps-script.js` o `appsscript.json`, actualizar también el proyecto de Google Apps Script **Citas JessicaOcampoFisio** y crear una nueva versión de la implementación web antes de dar la tarea por terminada.
- Después del despliegue, comprobar que la URL de producción responde y que el panel puede consultar el backend.
- Si Google solicita autorización, inicio de sesión o confirmación de permisos, detenerse únicamente para que Jessica complete ese paso y continuar después.
- No cambiar la URL pública del backend en `admin.html` salvo que una nueva implementación obligue a hacerlo.
